# Mongo-Scraper

Mongo-Scraper is a web app that scrapes news from another site. User can save articles, comment saved article, delete comment and delete saved article.

## Technologies used:
* JavaScript
* MongoDb
* EJS
* jQuery
* Bootstrap 3
* Node.js 
* Express npm package
* Request npm package
* Cheerio npm package
* Body - parser npm package
* mongojs npm package
